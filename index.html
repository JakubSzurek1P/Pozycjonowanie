Pozycjonowanie Elementów w CSS

Zrozum position i przejmij kontrolę nad układem strony.

Static Relative Absolute Fixed Sticky

1. Position: Static

To ustawienie domyślne. Element znajduje się w normalnym przepływie dokumentu. Właściwości top, right, bottom, left nie działają.

Jestem static (domyślny)

2. Position: Relative

Element jest pozycjonowany względem swojego pierwotnego położenia. Przesunięcie go nie wpływa na pozycję sąsiednich elementów – zostawia po sobie "puste miejsce".

Przesunięty o 20px w dół i w prawo

3. Position: Absolute

Element zostaje usunięty z normalnego przepływu i pozycjonowany względem najbliższego przodka, który ma ustawioną pozycję inną niż static.

Jestem w prawym dolnym rogu ramki

4. Position: Fixed

Element jest pozycjonowany względem okna przeglądarki (viewportu). Pozostaje w tym samym miejscu nawet podczas przewijania strony.

Spójrz w prawy dolny róg ekranu – widzisz pomarańczowe kółko?

FIXED

5. Position: Sticky

Hybryda relative i fixed. Element zachowuje się jak relative, dopóki nie osiągnie określonego punktu (np. góry ekranu), wtedy "przykleja się" jak fixed.

Nasz pasek nawigacji na górze strony używa właśnie position: sticky;
